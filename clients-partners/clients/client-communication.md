# Client Communication

{: .highlight }
> 고객과의 효과적인 소통 - 신뢰를 쌓고 프로젝트를 성공으로 이끄는 커뮤니케이션 원칙

## 개요

K-pop 의상 제작 업계에서 커뮤니케이션은 단순히 정보를 주고받는 것 이상입니다. 타이트한 일정, 높은 품질 기대, 창의적 비전과 현실적 제약 사이의 균형을 맞추는 과정에서, 어떻게 소통하느냐가 프로젝트의 성패를 좌우합니다.

이 가이드는 Kyndof 직원들이 고객과 소통할 때 따라야 할 원칙, 채널별 가이드라인, 상황별 대응 방법을 다룹니다.

### 왜 커뮤니케이션이 중요한가?

**신뢰 구축의 기반**
고객은 수천만 원에서 수억 원의 예산을 우리에게 맡깁니다. 그들이 우리를 신뢰하는 이유는 우리의 기술력뿐만 아니라, 투명하고 신속한 커뮤니케이션 때문입니다.

**오해와 갈등 예방**
K-pop 의상은 주관적 판단이 많이 개입되는 영역입니다. "화려한", "세련된", "과감한" 같은 단어는 사람마다 다르게 해석됩니다. 명확한 커뮤니케이션이 없으면 불필요한 수정과 갈등이 반복됩니다.

**일정 지연 방지**
컴백 일정은 고정되어 있습니다. 커뮤니케이션 지연으로 인한 시간 손실은 곧 야근과 품질 저하로 이어집니다.

**장기 파트너십 형성**
한 번의 프로젝트를 잘 마무리하는 것도 중요하지만, 지속적인 협업으로 이어지려면 신뢰할 수 있는 커뮤니케이션 파트너로 인식되어야 합니다.

## 핵심 커뮤니케이션 원칙

### 1. 신속성 (Speed)

**24시간 이내 1차 응답 원칙**

고객의 모든 문의는 24시간 이내에 1차 응답을 합니다. 설령 즉시 답을 줄 수 없더라도, "확인했고 언제까지 회신하겠습니다"라는 간단한 응답만으로도 고객은 안심합니다.

**긴급 요청의 우선순위화**

- **긴급**: 시상식/방송 일정이 48시간 이내 → 즉시 대응
- **중요**: 컴백 일정이 2주 이내 → 당일 대응
- **일반**: 신규 문의, 견적 요청 → 24시간 이내 대응

**자동 부재 알림 설정**

휴가, 출장 등으로 즉시 응답이 어려울 때는 이메일 자동 응답과 카톡 프로필 상태 메시지를 설정하고, 대체 연락처를 안내합니다.

### 2. 투명성 (Transparency)

**좋은 소식과 나쁜 소식 모두 공유**

문제가 발생했을 때 숨기거나 미루지 않습니다. 조기에 투명하게 공유하면 함께 해결 방안을 찾을 수 있지만, 늦게 알리면 신뢰가 무너집니다.

**진행 상황의 정기적 업데이트**

프로젝트가 순조롭게 진행되더라도, 정기적으로 진행 상황을 업데이트합니다. 침묵은 불안을 만듭니다.

예시:
```
[주간 업데이트]
안녕하세요, 스타일리스트님.
이번 주 진행 상황을 공유드립니다.

✅ 완료: 메인 무대 의상 3벌 제작 완료
🔄 진행 중: 서브 무대 의상 2벌 (금요일 완성 예정)
📅 예정: 다음 주 월요일 1차 피팅

일정에 변화 없이 순조롭게 진행 중입니다.
궁금하신 점 있으시면 언제든 연락 주세요!
```

**비용 투명성**

견적은 항목별로 명확히 제시하고, 추가 비용이 발생할 가능성이 있으면 사전에 알립니다.

### 3. 명확성 (Clarity)

**전문 용어 피하기**

고객이 패션 전문가가 아닐 수 있습니다. "다트", "바이어스", "오버록" 같은 용어는 사진과 함께 쉽게 설명합니다.

**구체적 수치와 일정**

❌ "곧 완성됩니다" → ✅ "이번 주 금요일 오후 5시까지 완성됩니다"
❌ "약간 추가 비용이 발생합니다" → ✅ "스팽글 추가로 50만 원의 추가 비용이 발생합니다"

**시각 자료 활용**

설명보다 사진 한 장이 효과적입니다. 원단 샘플, 디자인 스케치, 진행 과정 사진을 적극 활용합니다.

### 4. 공감과 존중 (Empathy & Respect)

**고객의 압박 이해하기**

스타일리스트도, 기획사 담당자도 우리만큼 압박받고 있습니다. 그들의 긴급함이나 불안을 개인적으로 받아들이지 않고, 함께 해결할 파트너로서 접근합니다.

**창의적 비전 존중**

"그건 안 됩니다"보다 "이런 방법으로 하면 가능합니다"를 먼저 생각합니다. 고객의 창의적 아이디어를 존중하면서도, 현실적 제약을 투명하게 설명합니다.

**문화적 감수성**

K-pop 업계는 빠르고 직설적인 커뮤니케이션을 선호합니다. 형식적인 인사보다 핵심을 빠르게 전달하되, 존중의 톤을 유지합니다.

## 채널별 커뮤니케이션 가이드

### 카카오톡

**언제 사용:**
- 긴급 문의 및 즉각적 응답이 필요한 경우
- 간단한 확인 및 업데이트
- 이미지/동영상 공유
- 일상적인 프로젝트 진행 관리

**가이드라인:**
- 업무 시간(09:00-18:00) 외에는 긴급한 경우에만 발송
- 긴 내용은 이메일로, 짧은 확인은 카톡으로
- 중요한 결정사항은 카톡 후 이메일로 재확인
- 읽음 표시 후 가능한 한 빨리 응답
- 이모티콘은 관계에 따라 적절히 사용 (친밀한 관계에서는 OK, 신규 고객에게는 자제)

**예시:**
```
[좋은 예]
안녕하세요! 요청하신 원단 샘플 사진 보내드립니다.
(사진)
A, B, C 중 선호하시는 것 알려주시면 진행하겠습니다!

[나쁜 예]
원단 어떤 거 하실 건가요? ㅎㅎ 빨리 좀 알려주세요~
```

### 이메일

**언제 사용:**
- 공식적인 견적, 계약서, 송장 발송
- 프로젝트 킥오프 및 최종 납품 통보
- 주간/월간 진행 상황 리포트
- 중요한 의사결정 기록 보존
- 여러 이해관계자가 참여하는 커뮤니케이션

**가이드라인:**
- 제목은 구체적으로: "의상 견적 요청" → "BLACKPINK 컴백 무대 의상 견적서 (2026.03.15 방송)"
- 첫 문단에 핵심 메시지 (결론부터 쓰기)
- 중요한 내용은 굵게, 일정/금액은 명확히
- 첨부 파일은 설명과 함께 (예: "첨부: 견적서, 디자인 스케치 3종")
- 서명에 연락처와 직책 명시

**이메일 템플릿 예시:**

```
제목: [Kyndof] ABC 컴백 무대 의상 최종 견적서

ABC 스타일리스트님, 안녕하세요.
Kyndof 김지은입니다.

요청하신 ABC 컴백 무대 의상 견적을 보내드립니다.

[핵심 정보]
- 총 금액: 4,500만 원 (VAT 별도)
- 제작 기간: 3주 (2026.02.15 - 03.08)
- 납품 일정: 2026.03.08 (금) 오후 3시
- 1차 피팅: 2026.02.25 (월) 오후 2시

상세 내역은 첨부된 견적서를 참고해주세요.
궁금하신 점 있으시면 언제든 연락 주세요!

첨부: 견적서_ABC컴백_20260215.pdf

감사합니다.

김지은
Account Manager, Kyndof
📧 jieun@kyndof.com
📱 010-1234-5678
```

### 전화/화상 미팅

**언제 사용:**
- 복잡한 디자인 콘셉트 논의
- 갈등 해결 및 긴급 상황 대응
- 프로젝트 킥오프 미팅
- 중요한 의사결정 (큰 계약, 일정 변경 등)

**가이드라인:**
- 사전에 아젠다 공유
- 미팅 후 핵심 내용 요약 이메일 발송 (기록 보존)
- 녹음은 상대방 동의 후에만
- 시간 약속 엄수 (K-pop 업계는 초 단위로 바쁩니다)

### 대면 미팅

**언제 사용:**
- 피팅 세션
- 원단 샘플 확인
- 신규 고객 첫 미팅
- 갈등 해결 및 관계 회복

**가이드라인:**
- 항상 준비물 체크리스트 작성
- 시간 약속 철저히 준수 (15분 일찍 도착)
- 미팅 후 감사 메시지 전송
- 약속한 후속 조치는 24시간 이내 시작

## 상황별 커뮤니케이션 시나리오

### 시나리오 1: 일정 지연 발생

**상황:** 원단 수급 문제로 납기가 3일 지연될 것 같습니다.

❌ **나쁜 대응:**
(침묵하거나, 납기 직전에 알림)

✅ **좋은 대응:**
```
스타일리스트님, 긴급히 알려드립니다.

해외 수입 원단 통관이 지연되어, 당초 약속드린
3월 8일 납품이 3월 11일로 변경될 것 같습니다.

[대안]
1. 국내 유사 원단으로 대체 (품질 95% 수준, 즉시 진행 가능)
2. 일부 의상만 원래 일정에 맞춰 먼저 납품
3. 지연된 일정 수용 (3일 늦춰서라도 원래 원단 사용)

무대 일정이 타이트한 것 알고 있어서 최대한 빨리 결정하셔야
일정을 맞출 수 있습니다. 오늘 중으로 말씀 주시면
즉시 진행하겠습니다.

죄송합니다. 바로 연락 주세요!
```

### 시나리오 2: 예산 초과 요청

**상황:** 고객이 예산을 초과하는 디자인을 요청했습니다.

❌ **나쁜 대응:**
"그건 예산 초과라서 안 됩니다."

✅ **좋은 대응:**
```
스타일리스트님, 요청하신 디자인 정말 멋집니다!

다만 스와로브스키 크리스탈을 전체에 적용하면
예산이 약 800만 원 초과됩니다.

[대안]
1. 앞면만 스와로브스키, 뒷면은 국산 크리스탈
   → 예산 내 가능, 무대 효과 90% 유지

2. 크리스탈 밀도를 70%로 조정
   → 예산 내 가능, 시각적 차이 거의 없음

3. 예산 증액 (800만 원)
   → 원래 디자인 그대로 진행

어떤 방향이 좋으실지 말씀 주세요!
샘플 보여드리면서 설명드려도 좋습니다.
```

### 시나리오 3: 불만 제기

**상황:** 고객이 피팅 결과에 불만을 표현했습니다.

❌ **나쁜 대응:**
"처음 주신 사이즈대로 만든 건데요..."

✅ **좋은 대응:**
```
말씀 감사합니다. 기대에 미치지 못해 죄송합니다.

구체적으로 어떤 부분이 불편하신지 들려주시겠어요?
- 사이즈 문제인지
- 디자인 방향 문제인지
- 소재 느낌 문제인지

정확히 이해해서 바로 수정하겠습니다.
무대 일정 고려해서 최우선으로 처리하겠습니다.

지금 당장 시간 되시면 다시 한번 뵙고
정확히 파악하고 싶습니다. 가능하신가요?
```

### 시나리오 4: 신규 고객 첫 미팅

**목표:** 신뢰 구축 + 요구사항 정확히 파악

**준비물:**
- Kyndof 포트폴리오
- 과거 유사 프로젝트 사례
- 원단 샘플북
- 프로세스 설명 자료

**대화 흐름:**
1. **경청**: 고객의 비전을 충분히 듣기 (80% 듣기, 20% 말하기)
2. **질문**: 구체적 요구사항 파악 (예산, 일정, 콘셉트, 우선순위)
3. **제안**: 우리의 강점과 과거 성공 사례 공유
4. **다음 단계**: 명확한 타임라인과 후속 조치 합의

## 커뮤니케이션 체크리스트

모든 중요한 고객 커뮤니케이션 전에 다음을 확인하세요:

**발송 전:**
- [ ] 팩트가 정확한가? (일정, 금액, 사양 재확인)
- [ ] 수신자가 맞는가?
- [ ] 톤이 적절한가? (존중하되 명확하게)
- [ ] 다음 액션이 명확한가? (누가, 무엇을, 언제까지)
- [ ] 첨부 파일이 있으면 정말 첨부했는가?

**발송 후:**
- [ ] 24시간 내 응답 없으면 팔로우업
- [ ] 중요한 내용은 CRM에 기록
- [ ] 약속한 후속 조치는 캘린더에 등록

## 금기 사항

절대 하지 말아야 할 커뮤니케이션:

❌ 고객 앞에서 다른 고객 이야기 (기밀 유지)
❌ "바빠서 못 봤어요" (변명보다 해결책)
❌ 확인 안 된 정보 전달 ("아마도", "~인 것 같아요")
❌ 감정적 반응 (비난받아도 프로페셔널하게)
❌ 약속 번복 (한 번 한 약속은 반드시 지키거나 사전 협의)
