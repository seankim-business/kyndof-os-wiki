# Governance

책임 체계와 의사결정 구조를 정의합니다.

## Contents

### RABSIC Framework
| Role | Description |
|------|-------------|
| **R** - Responsible | 실제 작업 수행자 |
| **A** - Accountable | 최종 책임자 (1명) |
| **B** - Backup | 백업 담당자 |
| **S** - Support | 지원 제공자 |
| **I** - Informed | 정보 수신자 |
| **C** - Consulted | 자문 제공자 |

### Policies
- 의사결정 정책
- 승인 절차
- 에스컬레이션 경로

### Decisions
- 아키텍처 결정 기록 (ADR)
- 주요 의사결정 히스토리
