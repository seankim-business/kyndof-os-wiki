# SOP: Pattern Creation

**Purpose:** Create accurate, production-ready patterns

## Steps

1. **Verify Measurements** - Confirm all measurements are current
2. **Select Base Pattern** - Choose commercial pattern or draft custom
3. **Modify for Design** - Adjust for design features (neckline, sleeves, etc.)
4. **Add Seam Allowances** - Standard 5/8" unless specified
5. **Label All Pieces** - Number, grain line, notches
6. **Create Mockup** - Test fit with muslin if complex
7. **Fit Test** - Verify fit on client or dress form
8. **Finalize Pattern** - Make any necessary adjustments
