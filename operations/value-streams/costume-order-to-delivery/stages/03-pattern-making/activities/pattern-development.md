# Activity: pattern development

**Owner:** Pattern Maker

## Purpose

Draft pattern pieces that translate design into construction-ready templates

## Process

### Standard Measurements Required

1. Select base pattern or draft from scratch
2. Modify for design features
3. Add seam allowances
4. Create mockup if complex
5. Fit test mockup
6. Adjust pattern as needed
