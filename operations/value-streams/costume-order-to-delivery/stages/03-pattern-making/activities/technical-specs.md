# Activity: technical specs

**Owner:** Pattern Maker

## Purpose

Document cutting and construction instructions for production team

## Process

### Standard Measurements Required

1. Number all pattern pieces
2. Create cut list with quantities
3. Mark grain lines and notches
4. Document seam allowances
5. Write construction sequence
6. Note special techniques required
