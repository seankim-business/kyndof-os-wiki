# Activity: measurement collection

**Owner:** Pattern Maker

## Purpose

Collect comprehensive measurements to ensure accurate fit

## Process

### Standard Measurements Required

- Bust/Chest circumference
- Waist circumference
- Hip circumference
- Shoulder width
- Arm length
- Inseam
- Torso length
- Neck circumference
- Additional measurements per garment type
