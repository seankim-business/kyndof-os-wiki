# Activity: quality check

**Owner:** Production Team

## Purpose

Verify costume meets quality standards before delivery

## Process

1. Check all seams for security
2. Verify embellishment is secure
3. Test all closures (zippers, buttons, snaps)
4. Check fit against measurements
5. Inspect for loose threads, raw edges
6. Verify matches approved design
7. Document any issues found
8. Pass/fail decision
