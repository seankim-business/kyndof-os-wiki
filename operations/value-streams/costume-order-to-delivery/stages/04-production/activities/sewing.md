# Activity: sewing

**Owner:** Production Team

## Purpose

Assemble garment following construction sequence

## Process

1. Follow construction sequence from pattern notes
2. Pin or baste before stitching
3. Use appropriate stitch type for fabric
4. Press seams as you go
5. Install closures (zippers, buttons, hooks)
6. Attach lining if applicable
7. Check fit periodically
