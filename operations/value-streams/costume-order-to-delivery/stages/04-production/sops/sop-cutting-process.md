# SOP: cutting process

**Purpose:** Ensure accurate, efficient fabric cutting

## Steps

1. **Pre-Flight Check** - Verify fabric quantity, pattern pieces complete
2. **Fabric Prep** - Pre-wash if needed, press flat
3. **Layout** - Arrange pattern pieces for efficient cutting
4. **Mark** - Trace pattern onto fabric, mark notches
5. **Cut** - Use sharp tools, cut accurately
6. **Label** - Tag each piece with number and orientation
7. **Quality Check** - Verify all pieces cut, markings transferred
