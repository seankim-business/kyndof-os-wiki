# Stage 4: Production

**Duration:** 7-14 days
**Owner:** Production Manager
**Trigger:** Approved pattern from Pattern Making
**Completion:** Quality-inspected costume ready for delivery

## Purpose

Fabricate the costume to specification with quality control at each step.

## Process Flow

```
Approved Pattern
     │
     ▼
Cutting (1-2 days) ──────▶ Fabric Layout + Cutting
     │
     ▼
Sewing & Assembly (3-7 days) ──────▶ Construction
     │
     ▼
Finishing & Embellishment (2-4 days) ──────▶ Details
     │
     ▼
Quality Inspection (1 day) ──────▶ [PASS] ──────▶ Delivery Stage
     │
     ▼
[FAIL] ──────▶ Rework ──────▶ Re-inspect
```

## Inputs

| Input | Source | Required Information |
|-------|--------|---------------------|
| Final Pattern | Pattern Maker | All pattern pieces |
| Cut List | Pattern Maker | Quantities, fabric requirements |
| Materials | Design Development | All fabrics, trims, notions |
| Construction Notes | Pattern Maker + Creative Director | Assembly sequence, techniques |

## Activities

### [1. Cutting](activities/cutting.md) (1-2 days)
Layout fabric, cut all pieces, mark notches and grain lines.

### [2. Sewing & Assembly](activities/sewing.md) (3-7 days)
Construct garment following pattern sequence.

### [3. Finishing](activities/finishing.md) (2-4 days)
Add embellishment, hem, press, final details.

### [4. Quality Check](activities/quality-check.md) (1 day)
Inspect against quality checklist, verify fit.

## Outputs

| Output | Format | Delivered To |
|--------|--------|-------------|
| Completed Costume | Physical | Client Relations Manager |
| Quality Checklist | Document | Production Manager + Client Relations |
| Production Photos | Digital | Portfolio + Client |

## RABSIC Matrix

| Role | Position | Specific Responsibility |
|------|----------|------------------------|
| **A** (Accountable) | Production Manager | Owns production quality and timeline |
| **R** (Responsible) | Production Team | Executes cutting, sewing, finishing |
| **S** (Support) | Production Assistant | Prepares materials, assists with complex steps |
| **C** (Consulted) | Creative Director | Advises on embellishment placement, finishing details |
| **I** (Informed) | Client | Receives progress photos at milestones |

## Quality Gate

**Pass Criteria:**
- [ ] All seams secure and finished
- [ ] Embellishment matches design
- [ ] No loose threads or raw edges
- [ ] Fits measurements (verified on form or client)
- [ ] Closures function properly
- [ ] Matches approved design
