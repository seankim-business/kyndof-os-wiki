# Activity: packaging

**Owner:** Client Relations Manager

## Purpose

Package costume professionally for delivery

## Process

1. **Clean Inspection** - Final check for lint, loose threads
2. **Fold Carefully** - Use tissue paper to prevent creasing
3. **Package** - Garment bag or sturdy box
4. **Include Documents**:
   - Care instructions (printed)
   - Contact info for support
   - Satisfaction survey link
5. **Seal** - Professional presentation
