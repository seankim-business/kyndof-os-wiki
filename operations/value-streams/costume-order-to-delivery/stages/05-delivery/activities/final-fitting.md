# Activity: final fitting

**Owner:** Client Relations Manager

## Purpose

Verify fit and make minor adjustments if needed

## Process

1. **Schedule Session** - Coordinate time with client (60-90 min slot)
2. **Prepare Space** - Clean fitting room, good lighting, full-length mirror
3. **Client Tries On** - Assist with putting costume on properly
4. **Assess Fit** - Check all areas (shoulders, bust, waist, length)
5. **Minor Adjustments** - Pin any needed changes (<2 hours work)
6. **Take Photos** - Document final fit for portfolio
7. **Explain Care** - Walk through care instructions
8. **Confirm Satisfaction** - Ask if client is happy with result
