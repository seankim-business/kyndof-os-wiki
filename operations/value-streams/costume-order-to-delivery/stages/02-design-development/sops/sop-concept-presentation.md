# SOP: Concept Presentation

**Purpose:** Present design concepts to clients in a way that facilitates decision-making

## Steps

1. **Prepare Lookbook** - Organize all concepts in single PDF/presentation
2. **Set Context** - Remind client of their brief requirements
3. **Present Each Concept** - 5-10 minutes per concept, explain rationale
4. **Facilitate Discussion** - Ask open-ended questions about preferences
5. **Guide Decision** - Help client choose or request hybrid
6. **Document Approval** - Get signature on selected concept
