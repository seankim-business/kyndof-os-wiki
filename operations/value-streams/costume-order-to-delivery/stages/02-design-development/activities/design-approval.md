# Activity: Design Approval

**Duration:** 2 days
**Owner:** Creative Director
**Output:** Client-signed approval with selected concept

## Presentation Process

### 1. Prepare Lookbook (Day 1)
Create professional presentation showing:
- All 3-5 concepts side-by-side
- Material swatches for each
- Cost and timeline for each
- Construction approach notes

### 2. Client Meeting (Day 1)
- Present each concept (5-10 min each)
- Answer questions
- Gather feedback
- Request selection or hybrid

### 3. Refinement (Day 2)
- If approved: Finalize materials
- If hybrid requested: Combine elements
- If revision needed: Adjust and re-present

### 4. Final Approval
Client signs approval form confirming:
- Selected design concept
- Material specifications
- Color palette
- Timeline acknowledgment
- Cost acknowledgment
