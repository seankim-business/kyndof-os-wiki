# Value Stream Stages

The Costume Order to Delivery value stream consists of 5 sequential stages, each with defined inputs, outputs, activities, and quality gates.

## All Stages

### [01. Order Reception](01-order-reception/) (1 day)
**Purpose:** Capture client requirements and establish project parameters
**Owner:** Client Relations Manager
**Output:** Documented brief + timeline + signed agreement

### [02. Design Development](02-design-development/) (5-7 days)
**Purpose:** Create and approve costume design concept
**Owner:** Creative Director
**Output:** Approved design + material list + color palette

### [03. Pattern Making](03-pattern-making/) (3-5 days)
**Purpose:** Translate design into technical construction patterns
**Owner:** Pattern Maker
**Output:** Final pattern + cut list + construction notes

### [04. Production](04-production/) (7-14 days)
**Purpose:** Fabricate the costume to specification
**Owner:** Production Manager
**Output:** Completed costume + quality checklist

### [05. Delivery](05-delivery/) (1-2 days)
**Purpose:** Ensure fit and deliver to client
**Owner:** Client Relations Manager
**Output:** Delivered costume + care guide + satisfaction survey

## Stage Dependencies

Each stage builds on the outputs of the previous stage:

```
Reception Output → Design Input
Design Output → Pattern Input
Pattern Output → Production Input
Production Output → Delivery Input
```

## Quality Gates

Every stage must pass a quality gate before proceeding:

| Stage | Gate | Criteria |
|-------|------|----------|
| Reception | Brief Complete | All requirements documented, timeline agreed |
| Design | Client Approval | Client signs off on design concept |
| Pattern | Fit Verified | Pattern matches measurements, mockup approved |
| Production | Quality Pass | Costume passes inspection checklist |
| Delivery | Client Acceptance | Client satisfaction score >8/10 |

## RABSIC Matrix by Stage

Each stage has its own accountability structure:

| Stage | Accountable (A) | Responsible (R) | Support (S) | Consulted (C) | Informed (I) |
|-------|----------------|-----------------|-------------|---------------|--------------|

| Design | Creative Director | Creative Director | Design Asst | Pattern Maker | Client |
| Pattern | Pattern Maker | Pattern Maker | Production Asst | Creative Director | Production Mgr |
| Production | Production Manager | Production Team | Production Asst | Creative Director | Client |
| Delivery | Client Relations Mgr | Client Relations Mgr | Production Asst | Production Mgr | Creative Director |
