## Directory Structure Pattern

Each function follows this pattern:

```
{function-name}/
├── index.md                    # Function overview
├── processes/
│   ├── index.md               # Processes list
│   └── {process-name}/
│       ├── index.md           # Process overview with RABSIC matrix
│       ├── steps/
│       │   ├── index.md       # Steps list
│       │   ├── 01-{step}.md   # Detailed step instructions
│       │   ├── 02-{step}.md
│       │   └── ...
│       └── sops/
│           ├── index.md       # SOPs list
│           ├── sop-{name}.md  # Detailed SOP
│           └── ...
└── responsibilities.md         # RABSIC matrix
```
