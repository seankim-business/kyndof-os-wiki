# Hiring Process

## Overview
Full recruitment lifecycle from job posting through employee onboarding.

**Duration:** 3-6 weeks (varies by role level)
**Owner:** HR (Recruiter)
**Tool:** Notion Candidate Tracker, BambooHR
**Trigger:** Approved headcount and job requisition

## Objectives
- Attract qualified candidates aligned with role requirements and company culture
- Conduct thorough, unbiased evaluation
- Deliver exceptional candidate experience
- Achieve high offer acceptance and employee retention

| Decision/Task | R | A | B | S | I | C |
|---------------|---|---|---|---|---|---|
| Job description creation | Recruiter | HR Director | - | Hiring Manager | - | Hiring Manager |
| Job posting and sourcing | Recruiter | Recruiter | HR Coordinator | - | HR Director | - |
| Resume screening | Recruiter | Recruiter | - | - | Hiring Manager | - |
| Phone screening | Recruiter | Recruiter | - | - | Hiring Manager | - |
| Interview coordination | HR Coordinator | Recruiter | Recruiter | - | - | - |
| Final candidate selection | Hiring Manager | Hiring Manager | - | HR Director | - | Recruiter, Team Members |
| Offer approval | HR Director | HR Director | - | Finance | CEO (senior roles) | - |
| Offer extension | Recruiter | HR Director | - | - | Hiring Manager | - |
| Onboarding coordination | HR Coordinator | HR Director | Recruiter | IT, Facilities | Hiring Manager | - |

## Success Criteria
- Time to hire ≤30 days (from posting to offer acceptance)
- Offer acceptance rate ≥80%
- New hire retention ≥90% at 90 days
- Candidate experience rating ≥4/5
- Hiring manager satisfaction ≥4/5

## Metrics Tracked
- Time to fill (by role type)
- Candidate pipeline conversion rates
- Source effectiveness (job boards, referrals, LinkedIn)
- Interview-to-offer ratio
- Offer acceptance rate
- 90-day retention rate
- Cost per hire
