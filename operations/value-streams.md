# Value Streams

{: .highlight }
> 고객에게 가치를 전달하는 End-to-End 프로세스

## 핵심 가치 흐름

### 1. 아티스트 의상 제작 (B2B)

```
┌──────────────────────────────────────────────────────────────────┐
│                    ARTIST COSTUME PRODUCTION                      │
├──────────────────────────────────────────────────────────────────┤
│                                                                   │
│  ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐          │
│  │ Request │ → │ Design  │ → │ Produce │ → │ Deliver │          │
│  └─────────┘   └─────────┘   └─────────┘   └─────────┘          │
│       │            │             │             │                  │
│       ▼            ▼             ▼             ▼                  │
│   - 브리핑      - 시안 개발   - 패턴/봉제   - 피팅             │
│   - 레퍼런스    - 컨펌        - 수정        - 최종 조율         │
│   - 일정 협의   - 소싱        - 마감        - 배송              │
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```

**Lead Time 목표:** 최소화 (업계 최단)

| 단계 | 목표 기간 | 현재 | 상태 |
|:-----|:---------|:-----|:-----|
| Request → Design | 1-2일 | - | 📝 측정 중 |
| Design → Produce | 3-5일 | - | 📝 측정 중 |
| Produce → Deliver | 1-2일 | - | 📝 측정 중 |

## 개선 기회

{: .warning }
> **현재 Off Track 목표들이 Value Stream 병목과 연결되어 있습니다.**

| 병목 | 연결된 목표 | 개선 방향 |
|:-----|:-----------|:---------|
| 납기 지연 | 납기를 극단적으로 단축한다 | 프로세스 자동화 |
| 디자인 반복 | 디자인 워크플로우 혁신 | 템플릿화, 도구 개선 |
| 실물 구현 차이 | 100% 정확한 실물 구현 | 품질 체크포인트 강화 |
