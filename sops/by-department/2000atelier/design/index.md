## Process Overview

The design process transforms client vision into actionable creative direction for production teams.

### Design Workflow
```
Client Inquiry → [SOP-001] Design Brief Intake → [SOP-002] Concept Presentation →
Client Approval → [SOP-003] Revisions (if needed) → Approved Design → Pattern Team
```

## Key Roles

- **Creative Director:** Owns design quality, approves all projects
- **Design Coordinator:** Executes intake process, manages designer assignments
- **Lead Designer:** Creates concepts, presents to clients, implements revisions

## Quality Metrics

- **Brief Accuracy:** ≥95% (briefs requiring no post-approval amendments)
- **Client Approval Rate:** ≥90% (concepts approved on first presentation)
- **Intake Cycle Time:** ≤5 business days (inquiry to approved brief)
