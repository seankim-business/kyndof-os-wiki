# Vision & Mission

{: .highlight }
> 회사의 존재 이유와 방향성을 정의합니다.

## Vision

**우리가 만들고자 하는 세상**

> *"엔터테인먼트와 패션의 경계를 허물어, 모든 아티스트가 자신만의 스타일로 빛나는 세상"*

## Mission

**비전을 실현하기 위해 우리가 하는 일**

> *"최고의 아티스트 의상을 통해 K-컬처의 글로벌 영향력을 확장한다"*

## Core Values

### 1. Excellence (탁월함)
- 어떤 시안이든 100% 정확하게 실물 구현
- 고객 만족도를 극단적으로 끌어올림

### 2. Innovation (혁신)
- 디자인 워크플로우 혁신
- 새로운 기준 제시

### 3. Speed (속도)
- 납기를 극단적으로 단축
- 빠른 피드백 루프

### 4. Partnership (파트너십)
- 탑스타와의 긴밀한 협업
- 커뮤니티 기반 성장
