# Strategy

{: .highlight }
> **Layer 1**: 회사의 존재 이유와 방향성을 정의합니다.

회사의 전략적 기반을 형성하는 핵심 요소들입니다:

| 구성요소 | 설명 | 페이지 |
|:---------|:-----|:-------|
| **Vision & Mission** | 왜 존재하는가? 어디로 가는가? | [보기](vision-mission/) |
| **Goals** | 무엇을 달성할 것인가? (31개 목표) | [보기](goals-strategy/) |
| **Strategies** | 어떻게 달성할 것인가? (60개 전략) | [보기](goals-strategy/) |

## IMS 흐름에서의 위치

```
┌─────────────────────────────────────────────────┐
│  ★ STRATEGY (전략) ← You are here               │
│     Vision → Goals → Strategies                │
├─────────────────────────────────────────────────┤
│  2. OPERATIONS (운영)                            │
│     Business Model → Value Streams → Functions │
├─────────────────────────────────────────────────┤
│  3. EXECUTION (실행)                             │
│     Processes → Projects → Tasks               │
├─────────────────────────────────────────────────┤
│  4. GOVERNANCE (거버넌스)                         │
│     RABSIC → Policies → Decisions              │
└─────────────────────────────────────────────────┘
```
