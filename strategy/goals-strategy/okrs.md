# OKRs

{: .highlight }
> "측정할 수 없으면 관리할 수 없다. 관리할 수 없으면 개선할 수 없다."

## 우리가 추구하는 것

OKR(Objectives and Key Results)은 킨도프의 목표 관리 시스템입니다. Google, Intel, LinkedIn 등 세계적인 기업들이 사용하는 프레임워크로, 조직의 모든 구성원이 같은 방향을 바라보고 진행 상황을 측정할 수 있게 합니다.

이 문서는 OKR의 기본 개념과 킨도프에서의 적용 방법을 설명합니다.

---

## OKR 기본 개념

### Objective (목표)

**"무엇을 달성할 것인가?"**

Objective는 달성하고 싶은 것입니다. 정성적이고 영감을 주는 표현으로 작성합니다.

**좋은 Objective의 특징:**
- 행동을 유발하고 영감을 준다
- 명확한 방향성을 제시한다
- 도전적이지만 달성 가능하다
- 구체적인 기간이 있다

**예시:**
- "고객 만족도를 극단적으로 끌어올린다" (O)
- "매출 10% 증가" (X - 이것은 Key Result)

### Key Results (핵심 결과)

**"어떻게 달성 여부를 측정할 것인가?"**

Key Results는 Objective 달성 여부를 측정하는 정량적 지표입니다. 숫자로 표현되어야 합니다.

**좋은 Key Results의 특징:**
- 구체적이고 측정 가능하다
- 시작점과 목표점이 명확하다
- 3-5개 이내로 제한한다
- 결과 중심이다 (활동이 아님)

**예시:**
- "NPS 점수 60점 → 80점" (O)
- "고객 설문 실시" (X - 이것은 활동)

## 킨도프 OKR 구조

킨도프의 OKR은 4단계 계층 구조를 가집니다.

```
Company OKR (전사 목표)
    ↓
Brand OKR (브랜드별 목표: 아뜰리에, 아카이브)
    ↓
Team OKR (팀별 목표)
    ↓
Individual OKR (개인별 목표)
```

### 정렬 원칙

상위 OKR의 Key Result가 하위 OKR의 Objective가 됩니다.

**예시:**

| 레벨 | OKR |
|:-----|:----|
| **Company** | O: 엔터-패션 브랜드 1위 |
| | KR1: 고객 만족도 90% 달성 |
| **Brand (Archives)** | O: 고객 만족도 90% 달성 |
| | KR1: NPS 80점 달성 |
| **Team (CS)** | O: NPS 80점 달성 |
| | KR1: 응대 시간 24시간 → 4시간 |
| **Individual** | O: 응대 시간 4시간 달성 |
| | KR1: 매일 오전/오후 문의 처리 완료 |

| 모델 커뮤니티 가입자 | 0 | 500명 | 멤버십 가입 |

## OKR 작성 가이드

### 작성 원칙

1. **도전적으로**: 70% 달성이면 성공. 100% 달성하면 목표가 쉬웠다는 뜻
2. **측정 가능하게**: 숫자가 없으면 Key Result가 아님
3. **결과 중심으로**: "~를 한다"가 아니라 "~가 된다"
4. **3-5개로 제한**: 많으면 집중이 안 됨

### 흔한 실수와 수정

| 실수 | 수정 |
|:-----|:-----|
| "고객 서비스 개선" | "NPS 점수 60 → 80" |
| "마케팅 캠페인 실행" | "캠페인당 구매 전환 500건" |
| "디자인 품질 향상" | "시안-실물 일치율 98%" |
| "팀원 교육 실시" | "팀원 스킬 평가 평균 4.5점/5점" |

### 체크리스트

Objective 작성 시:
- [ ] 영감을 주는가?
- [ ] 방향이 명확한가?
- [ ] 기간이 정해져 있는가?

Key Result 작성 시:
- [ ] 숫자로 측정 가능한가?
- [ ] 시작점과 목표점이 있는가?
- [ ] 결과인가? (활동이 아니라)
- [ ] 3-5개 이내인가?

## OKR 운영 주기

### 분기 주기

| 시점 | 활동 |
|:-----|:-----|
| **Q 시작 2주 전** | 다음 분기 OKR 초안 작성 |
| **Q 시작 1주 전** | OKR 정렬 미팅 (상하위 연결 확인) |
| **Q 시작** | OKR 확정 및 공유 |
| **매주** | Key Results 진행 상황 업데이트 |
| **Q 중간** | 중간 점검 및 조정 |
| **Q 종료** | 회고 및 점수 산정 |

### 점수 산정

| 점수 | 의미 |
|:-----|:-----|
| **0.0 - 0.3** | 실패. 목표 자체가 잘못됐거나 실행이 부족 |
| **0.4 - 0.6** | 진전은 있지만 목표 미달 |
| **0.7 - 0.9** | 성공. 도전적 목표에 근접 |
| **1.0** | 완벽 달성. (목표가 쉬웠을 수 있음) |

**0.7이 이상적인 점수입니다.** 매번 1.0이면 목표가 너무 쉽고, 매번 0.3이면 목표가 비현실적입니다.

## OKR과 킨도프 핵심 가치

OKR은 킨도프의 세 가지 핵심 가치와 연결됩니다.

### Position > Person
OKR은 역할(Position)에 부여됩니다. 담당자가 바뀌어도 OKR은 유지됩니다.

### System > Individual
OKR 자체가 시스템입니다. 개인의 감이 아니라 정량적 지표로 성과를 측정합니다.

### Sisyphus Mode
Key Result가 달성될 때까지 끝난 것이 아닙니다. 숫자로 검증됩니다.
