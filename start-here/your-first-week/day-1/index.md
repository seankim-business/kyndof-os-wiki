# 첫째 날

환영합니다. 오늘은 긴장되는 하루입니다.

**걱정하지 마세요. 천천히 하나씩 해봅시다.**

---

## 오전: 환영과 셋업

### 1. 환영 미팅 (30분)

매니저 또는 온보딩 담당자와 미팅이 있을 겁니다.

**이 미팅에서 나올 내용:**
- 자기소개 (간단하게)
- 오늘 일정 확인
- 계정 생성 순서 안내
- 자리 배정 및 장비 수령

**준비할 것:**
- 간단한 자기소개 (이름, 경력, 왜 킨도프에 왔는지)
- 궁금한 점 메모

### 2. 계정 생성 (1-2시간)

다음 계정들을 만들거나 초대를 받습니다:

#### 필수 계정

**회사 이메일**
- 형식: `이름@kyndof.com`
- 모든 다른 계정의 기본 이메일이 됨

**Slack**
- 회사 내부 커뮤니케이션
- 초대 링크는 이메일로 받음
- 가입 후 프로필 사진 설정

**Notion**
- 업무 문서, 프로젝트 관리
- 초대 링크는 이메일로 받음
- 회사 이메일로 가입

**GitHub**
- 코드 저장소 (개발자/기술직)
- 개인 계정으로 조직 초대 받음
- 2FA (2단계 인증) 설정 필수

#### 역할별 추가 계정

**디자이너:**
- Figma (디자인 협업)

**개발자:**
- Railway (배포 환경 - 필요시)

**체크리스트:**
- [ ] 회사 이메일 로그인 확인
- [ ] Slack 가입 및 프로필 설정
- [ ] Notion 초대 수락
- [ ] GitHub 조직 가입 (해당시)
- [ ] 추가 도구 계정 (역할별)

### 3. 장비 및 자리 (30분)

**받을 것들:**
- 노트북 (또는 데스크톱)
- 마우스, 키보드
- 모니터 (필요시)
- 기타 업무 도구

**자리 배정:**
- 팀 동료들 근처
- 필요한 것 (책상, 의자 조정) 요청

**격식 없이 편하게:**

## 오후: 읽고 이해하기

### 4. Wiki 읽기 (1-2시간)

이 Wiki의 핵심 섹션을 읽습니다:

**필수 읽기:**
- [우리는 누구인가](../../who-we-are/) - 전체 (30분)
  - [우리의 여정](../../who-we-are/our-story.md)
  - [우리가 가는 곳](../../who-we-are/where-we-are-going.md)
  - [우리가 믿는 것](../../who-we-are/what-we-believe.md)

**이해해야 할 것:**
- 킨도프의 비전 (LVMH급 글로벌 럭셔리 그룹)
- 두 개의 비즈니스 (Atelier + Archives)
- 핵심 철학 (복리의 기계, 눈덩이 굴리기)

**체크리스트:**
- [ ] "우리는 누구인가" 섹션 완독
- [ ] 킨도프의 비전을 한 문장으로 설명할 수 있음
- [ ] "복리의 기계"가 무슨 뜻인지 이해함
- [ ] 궁금한 점 메모

### 5. 팀 소개 미팅 (30분-1시간)

팀 전체 또는 일부와 미팅.

**목적:**
- 서로 얼굴 익히기
- 각자 하는 일 간단히 소개
- 협업 방식 이해

**준비할 질문:**
- "우리 팀의 가장 큰 목표가 뭔가요?"
- "팀에서 제가 배워야 할 것은 뭔가요?"
- "협업할 때 주의할 점이 있나요?"

### 6. Slack에 자기소개 (15분)

`#general` 채널에 자기소개 메시지 올리기.

**템플릿:**

```
안녕하세요, 오늘부터 [팀 이름]에서 [역할]로 일하게 된 [이름]입니다.

[간단한 경력 또는 배경 1-2줄]

킨도프에서 [무엇을 하고 싶은지/기대하는지] 기대됩니다.
잘 부탁드립니다!
```

**예시:**

```
안녕하세요, 오늘부터 Brand Ops팀에서 마케터로 일하게 된 김지민입니다.

이전에는 패션 스타트업에서 퍼포먼스 마케팅을 2년 했고,
K-pop과 패션의 교차점에서 일하고 싶어 킨도프에 합류했습니다.

많이 배우고 기여하겠습니다. 잘 부탁드립니다!
```

## 첫날 FAQ

### "너무 정보가 많아요."

맞습니다. 한 번에 다 이해할 필요 없습니다.

**오늘 목표:**
- 계정 만들기 ✅
- 팀원 얼굴 익히기 ✅
- 회사 비전 대략 이해하기 ✅

나머지는 천천히 배우면 됩니다.

### "Wiki를 다 못 읽었어요."

괜찮습니다. 첫날은 [우리는 누구인가](../../who-we-are/) 섹션만 읽으면 됩니다.

나머지는 이번 주 동안 천천히 읽으세요.

### "아무것도 안 한 것 같아요."

첫날은 "하는 날"이 아닙니다. **"이해하는 날"**입니다.

- 회사를 이해했습니다.
- 팀을 만났습니다.
- 환경을 셋업했습니다.

이미 충분히 했습니다.

### "내일은 뭘 하나요?"

내일부터는 팀원들과 1:1 미팅 + 작은 task 시작.

**매니저가 내일 일정을 안내할 겁니다.**
