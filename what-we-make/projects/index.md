# Projects (프로젝트)

{: .highlight }
> Kyndof가 수행하는 다양한 전략적, 운영적 프로젝트들을 소개합니다. 반복적인 의상 제작 외에 회사가 성장하고 혁신하기 위해 추진하는 모든 이니셔티브를 다룹니다.

## 개요

Kyndof는 고객을 위한 의상을 제작하는 것 외에도, 회사 자체를 발전시키기 위한 다양한 프로젝트를 수행합니다. 이러한 프로젝트들은 우리의 역량을 강화하고, 새로운 기회를 탐색하며, 더 나은 서비스를 제공할 수 있는 기반을 만듭니다.

프로젝트는 일상적인 의상 제작 업무와 구분됩니다. 의상 주문은 정해진 프로세스(주문 → 디자인 → 패턴 → 생산 → 배송)를 따르는 반복적인 작업입니다. 반면, 프로젝트는 특정 목표를 달성하기 위해 한시적으로 추진되는 활동입니다. 시작과 끝이 있고, 명확한 산출물을 목표로 합니다.

## 프로젝트와 일상 업무의 차이

| 구분 | 일상 업무 (의상 주문) | 프로젝트 |
|------|----------------------|----------|
| 기간 | 반복적, 지속적 | 한시적, 명확한 종료 |
| 프로세스 | 정해진 절차 따름 | 프로젝트별 고유 계획 |
| 산출물 | 의상 | 다양 (시스템, 문서, 역량 등) |
| 관리 | 운영 관리 | 프로젝트 관리 |
| 예시 | A사 무대 의상 제작 | ERP 시스템 도입 |

## 프로젝트 유형

Kyndof에서 수행하는 프로젝트는 크게 다음 유형으로 분류됩니다:

### 1. 전략 프로젝트 (Strategic)

회사의 장기적인 방향과 성장에 관련된 프로젝트입니다:

- 새로운 시장 진출
- 브랜드 포지셔닝
- 대형 파트너십 구축
- 비즈니스 모델 혁신

### 2. 운영 프로젝트 (Operational)

내부 프로세스와 시스템을 개선하는 프로젝트입니다:

- ERP/PLM 시스템 도입
- 프로세스 자동화
- 품질 관리 체계 강화
- 생산성 향상 이니셔티브

### 3. 실험 프로젝트 (Experiment)

새로운 아이디어나 접근법을 테스트하는 프로젝트입니다:

- 새로운 제품 라인 테스트
- 신기술 도입 검토
- 파일럿 프로그램
- 시장 검증

### 4. 지원 프로젝트 (Support)

회사 운영을 지원하는 지속적인 활동입니다:

- 정부 지원 사업 수행
- 기획전 및 프로모션
- 콘텐츠 제작
- 행정 업무 체계화

## 프로젝트 상태

모든 프로젝트는 다음 상태 중 하나를 가집니다:

| 상태 | 설명 |
|------|------|
| **In Progress** | 현재 적극적으로 진행 중인 프로젝트 |
| **Always On** | 종료 없이 지속적으로 운영되는 프로젝트 |
| **Done** | 목표를 달성하고 완료된 프로젝트 |
| **Cancelled** | 중단된 프로젝트 (사유 기록) |
| **On Hold** | 일시 중지된 프로젝트 |

## 프로젝트 관리 방식

### RABSIC 기반 책임 할당

모든 프로젝트에는 RABSIC 역할이 할당됩니다:

- **R (Responsible)**: 실제로 프로젝트 작업을 수행하는 사람
- **A (Accountable)**: 프로젝트 결과에 대한 최종 책임자 (1명)
- **S (Support)**: 프로젝트 수행을 지원하는 사람
- **C (Consulted)**: 결정 전에 의견을 구하는 전문가
- **I (Informed)**: 진행 상황을 통보받는 이해관계자

### 프로젝트 추적

프로젝트는 다음 도구와 시스템을 통해 추적됩니다:

- **Notion**: 프로젝트 보드, 태스크 관리, 문서화
- **회의**: 주간 프로젝트 리뷰
- **보고**: 월간 프로젝트 현황 보고

### 성과 측정

프로젝트 성공은 다음을 기준으로 평가합니다:

- 목표 달성 여부
- 일정 준수 여부
- 예산 준수 여부
- 품질 수준
- 이해관계자 만족도

## 현재 프로젝트 현황

Kyndof 프로젝트의 전반적인 현황입니다:

- **총 등록 프로젝트**: 177개
- **현재 진행 중**: 12개 (6.8%)
- **완료**: 118개 (66.7%)
- **취소**: 42개 (23.7%)

대부분의 프로젝트가 성공적으로 완료되었거나, 상황 변화에 따라 적절히 중단되었습니다. 현재 12개의 활성 프로젝트가 진행 중이며, 그 중 7개는 한시적 프로젝트, 5개는 상시 운영 프로젝트입니다.

## 활성 프로젝트 목록

## 프로젝트 제안

새로운 프로젝트를 제안하려면:

1. 프로젝트 필요성과 기대 효과 정리
2. 대략적인 일정과 리소스 예측
3. 담당 팀장 또는 임원에게 제안
4. 프로젝트 위원회 검토 (필요시)
5. 승인 후 Notion에 프로젝트 생성
