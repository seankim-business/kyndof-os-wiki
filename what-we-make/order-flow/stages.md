# Order Stages Detail (주문 단계 상세)

{: .highlight }
> 주문 처리의 5단계 각각에서 일어나는 활동, 필요한 입력, 생성되는 산출물을 상세히 설명합니다.

## 개요

이 문서는 [주문 흐름](index.md)에서 소개한 5단계를 깊이 있게 다룹니다. 각 단계에서 누가 무엇을 하는지, 어떤 정보가 필요한지, 무엇이 만들어지는지를 명확히 합니다.

Kyndof의 주문 프로세스는 [가치 흐름: 의상 주문-배송](../../operations/value-streams/costume-order-to-delivery/)에서 정의된 표준 프로세스를 따릅니다. 이 문서는 고객 관점에서 이해하기 쉽게 재구성한 버전입니다.

## Stage 1: 주문 접수 (Order Reception)

### 목적

고객의 요청을 정확히 이해하고, 프로젝트를 시작할 수 있는 상태로 만듭니다.

### 소요 시간

약 1일 (클라이언트 응답 속도에 따라 변동)

### 주요 활동

**1. 초기 연락 수신**

클라이언트가 이메일, 전화, 또는 소개를 통해 연락합니다. 담당자는 다음을 파악합니다:

- 누구를 위한 의상인가? (아티스트/그룹 정보)
- 어떤 용도인가? (무대, 뮤직비디오, 화보 등)
- 언제 필요한가? (촬영/공연 날짜)
- 어떤 컨셉인가? (분위기, 레퍼런스)

**2. 가능성 검토**

요청받은 일정 내에 완료가 가능한지 평가합니다:

- 현재 작업량 확인
- 필요 리소스 가용성 확인
- 특수 소재 조달 기간 고려

**3. 견적 및 일정 제안**

가능성이 확인되면 정식 제안서를 작성합니다:

- 예상 비용 범위
- 단계별 일정
- 필요한 클라이언트 참여 포인트

**4. 계약 체결**

양측이 조건에 합의하면:

- 계약서 서명
- 선금 수령 (일반적으로 30-50%)
- 프로젝트 공식 시작

### 필요 입력

| 정보 | 설명 | 필수 여부 |
|------|------|----------|
| 아티스트 정보 | 이름, 그룹, 인원수 | 필수 |
| 활동 유형 | 무대/MV/화보 등 | 필수 |
| 컨셉 설명 | 키워드, 레퍼런스 이미지 | 필수 |
| 필요 날짜 | 촬영/공연 일정 | 필수 |
| 예산 범위 | 대략적인 범위도 도움 | 권장 |
| 기존 자료 | 과거 의상 사진, 브랜드 가이드 | 선택 |

### 산출물

- 문서화된 요청서 (Brief)
- 견적서 및 일정표
- 서명된 계약서
- 커뮤니케이션 계획 (담당자 연락처, 미팅 일정)

### 담당자

| 역할 | 포지션 |
|------|--------|
| **R** (Responsible) | Client Relations Manager |
| **A** (Accountable) | Production Manager |

## Stage 2: 디자인 개발 (Design Development)

### 목적

클라이언트의 컨셉을 구체적인 의상 디자인으로 시각화하고 확정합니다.

### 소요 시간

5-7일 (피드백 속도와 수정 횟수에 따라 변동)

### 주요 활동

**1. 컨셉 리서치 (1-2일)**

디자인 팀이 요청서를 기반으로 깊이 있는 리서치를 수행합니다:

- 앨범 컨셉, 가사, 세계관 분석
- 아티스트의 과거 스타일 히스토리 검토
- 현재 트렌드 및 경쟁 분석
- 영감 보드(Mood Board) 제작

**2. 컨셉 스케치 (2-3일)**

3-5개의 디자인 옵션을 스케치합니다:

- 각 옵션별 전체 실루엣
- 주요 디자인 포인트
- 색상 제안
- 예상 소재

**3. 클라이언트 프레젠테이션**

스케치를 클라이언트에게 제시하고 피드백을 수집합니다:

- 선호 방향 확인
- 수정 요청 사항 기록
- 혼합 아이디어 논의 (A안의 상의 + B안의 하의 등)

**4. 디자인 수정 (1-2일)**

피드백을 반영하여 디자인을 수정합니다. 이 과정이 여러 번 반복될 수 있습니다.

**5. 최종 디자인 확정**

클라이언트가 최종 디자인을 승인합니다:

- 서면 승인 (이메일 또는 서명)
- 이후 디자인 변경 시 추가 비용/일정 발생 안내

**6. 소재 선정 및 조달 시작**

확정된 디자인에 맞는 원단과 부자재를 선택하고 조달을 시작합니다.

### 필요 입력

| 정보 | 설명 | 필수 여부 |
|------|------|----------|
| 승인된 요청서 | Stage 1에서 작성 | 필수 |
| 상세 레퍼런스 | 구체적인 참고 이미지 | 권장 |
| 피드백 | 스케치에 대한 의견 | 필수 |
| 최종 승인 | 디자인 확정 의사 | 필수 |

### 산출물

- 최종 디자인 스케치 (컬러, 다각도)
- 소재 스왓치 보드
- 색상 팔레트 문서
- 디자인 사양서 (패턴 팀 전달용)

### 담당자

| 역할 | 포지션 |
|------|--------|
| **R** (Responsible) | Lead Designer |
| **A** (Accountable) | Creative Director |
| **C** (Consulted) | 클라이언트 |

## Stage 3: 패턴 제작 (Pattern Making)

### 목적

디자인을 실제 옷으로 만들기 위한 기술적 설계도(패턴)를 완성합니다.

### 소요 시간

3-5일 (토일 제작 여부에 따라 변동)

### 주요 활동

**1. 치수 수집 (0.5일)**

아티스트의 정확한 신체 치수를 확보합니다:

- 직접 측정 (가장 정확)
- 기존 데이터 활용
- 스타일리스트 제공 치수 검증

**2. 패턴 제도 (1-2일)**

디자인 사양서를 기반으로 패턴을 제도합니다:

- 기본 블록 패턴 작성
- 디자인에 맞춘 스타일 변형
- 안감, 심지 패턴 추가

**3. 토일 제작 (선택, 1-2일)**

복잡한 디자인이나 핏이 중요한 경우:

- 저렴한 원단으로 시제품 제작
- 착용 테스트
- 수정 사항 파악

**4. 패턴 수정 (0.5-1일)**

토일 테스트나 검토 결과를 반영:

- 핏 조정
- 구조 개선
- 움직임 여유 추가

**5. 기술 사양서 작성**

생산 팀에 전달할 문서 완성:

- 봉제 순서 및 방법
- 주의사항
- 부자재 배치

### 필요 입력

| 정보 | 설명 | 필수 여부 |
|------|------|----------|
| 최종 디자인 | Stage 2에서 확정 | 필수 |
| 아티스트 치수 | 정확한 신체 측정값 | 필수 |
| 소재 정보 | 원단 특성 (신축성, 두께 등) | 필수 |

### 산출물

- 완성된 패턴 세트
- 재단 배치도 (Marker)
- 기술 사양서
- 원단 소요량 계산

### 담당자

| 역할 | 포지션 |
|------|--------|
| **R** (Responsible) | Pattern Maker |
| **A** (Accountable) | Pattern Making Lead |
| **C** (Consulted) | Creative Director (복잡한 구조 시) |

## Stage 4: 생산 (Production)

### 목적

패턴과 소재를 사용하여 실제 의상을 제작합니다.

### 소요 시간

7-14일 (복잡도와 장식 수준에 따라)

### 주요 활동

**1. 재단 (1-2일)**

원단을 패턴에 따라 자릅니다:

- 마커 배치 및 원단 펼침
- 정밀 재단
- 조각 번들링

**2. 봉제 (3-7일)**

재단된 조각을 조립합니다:

- 부분 제작 (소매, 칼라, 포켓 등)
- 본체 조립
- 안감 처리
- 지퍼, 단추 등 부착

**3. 장식 작업 (2-4일)**

디자인에 따른 장식 추가:

- 비즈, 스팽글, 스톤 수작업
- 자수, 아플리케
- 프린팅, 특수 가공

**4. 마감 (0.5일)**

최종 마무리:

- 실밥 정리
- 다림질, 프레싱
- 단추홀, 스냅 마감

**5. 품질 검사 (0.5일)**

완성된 의상 검수:

- 품질 체크리스트 확인
- 치수 측정
- 외관 및 기능 테스트

### 필요 입력

| 정보 | 설명 | 필수 여부 |
|------|------|----------|
| 완성 패턴 | Stage 3에서 확정 | 필수 |
| 기술 사양서 | 봉제 지시 | 필수 |
| 원부자재 | 조달 완료된 소재 | 필수 |

### 산출물

- 완성된 의상
- 품질 검사 보고서
- 관리 지침서

### 담당자

| 역할 | 포지션 |
|------|--------|
| **R** (Responsible) | Production Team |
| **A** (Accountable) | Production Manager |
| **S** (Support) | QC Inspector |

## Stage 5: 배송 (Delivery)

### 목적

완성된 의상을 클라이언트에게 전달하고 프로젝트를 마무리합니다.

### 소요 시간

1-2일

### 주요 활동

**1. 최종 피팅 (선택)**

가능한 경우 아티스트가 직접 착용:

- 핏 확인
- 움직임 테스트
- 미세 조정 필요 여부 파악

**2. 미세 조정 (필요시)**

최종 피팅에서 발견된 사항 수정:

- 기장 조절
- 허리 조절
- 장식 위치 미세 조정

**3. 포장**

의상을 안전하게 포장:

- 의류 커버
- 관리 지침 동봉
- 부속품 동봉 (여분 단추, 수선 키트 등)

**4. 배송 또는 수령**

클라이언트에게 전달:

- 직접 수령
- 배송 (보험 가입 권장)
- 현장 배달 (촬영장/공연장)

**5. 최종 확인**

프로젝트 마무리:

- 클라이언트 검수
- 수령 확인
- 만족도 조사
- 잔금 정산

### 필요 입력

| 정보 | 설명 | 필수 여부 |
|------|------|----------|
| 품질 검사 통과 의상 | Stage 4에서 완성 | 필수 |
| 배송 정보 | 수령 방법, 주소, 담당자 | 필수 |

### 산출물

- 전달된 의상
- 관리 지침서
- 수령 확인서
- 만족도 조사 결과

### 담당자

| 역할 | 포지션 |
|------|--------|
| **R** (Responsible) | Client Relations Manager |
| **A** (Accountable) | Production Manager |
| **I** (Informed) | Creative Director |
