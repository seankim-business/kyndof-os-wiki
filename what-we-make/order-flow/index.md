# Order Flow (주문 흐름)

{: .highlight }
> 고객 문의부터 완성된 의상 배송까지, Kyndof에서 주문이 어떻게 처리되는지 전체 흐름을 설명합니다.

## 개요

새로운 고객이 Kyndof에 연락하면 어떤 일이 일어날까요? 이 페이지는 주문이 접수되어 최종 배송되기까지의 전체 여정을 안내합니다. 이 흐름을 이해하면 각 단계에서 무엇을 기대할 수 있는지, 언제 결정을 내려야 하는지 명확해집니다.

Kyndof의 주문 처리는 철저한 프로세스를 따릅니다. 이 프로세스는 수년간의 경험을 통해 다듬어져, 품질과 일정을 모두 만족시킬 수 있도록 설계되었습니다. 무대 의상 제작은 긴급한 일정이 많은 분야입니다. 컴백 날짜는 고정되어 있고, 촬영 일정은 변경하기 어렵습니다. 그렇기 때문에 더욱 체계적인 프로세스가 필요합니다.

## 전체 흐름 개요

고객 문의에서 배송까지 5단계를 거칩니다:

```
[문의] → [디자인] → [패턴] → [생산] → [배송]
  1일      5-7일     3-5일    7-14일    1-2일
           ─────────────────────────────────
                    총 17-29일
```

각 단계는 이전 단계가 완료되어야 시작할 수 있습니다. 디자인이 확정되지 않으면 패턴을 만들 수 없고, 패턴이 없으면 생산에 들어갈 수 없습니다. 이 순차적인 구조는 품질을 보장하지만, 각 단계에서 지연이 발생하면 전체 일정에 영향을 미칩니다.

## 왜 이 프로세스인가?

### 품질 관문 (Quality Gates)

각 단계 사이에는 품질 관문이 있습니다. 다음 단계로 넘어가기 전에 반드시 확인하고 승인받아야 하는 항목들입니다:

| 관문 | 확인 내용 | 승인 주체 |
|------|-----------|-----------|
| 문의 → 디자인 | 요구사항 명확, 일정 합의, 계약 체결 | 클라이언트 & Kyndof |
| 디자인 → 패턴 | 디자인 승인, 소재 확정 | 클라이언트 |
| 패턴 → 생산 | 핏 확인 (필요시 토일 피팅) | Pattern Maker |
| 생산 → 배송 | 품질 검사 통과 | QC Inspector |
| 배송 완료 | 클라이언트 검수 및 수령 확인 | 클라이언트 |

이 관문들은 문제가 커지기 전에 조기에 발견하고 수정할 수 있게 해줍니다.

### 명확한 커뮤니케이션

각 단계에서 클라이언트가 알아야 할 것, 결정해야 할 것, 제공해야 할 것이 명확히 정의되어 있습니다. 이는 오해와 지연을 최소화합니다.

## 단계별 요약

### 1단계: 주문 접수 (Order Reception)

**소요 시간:** 약 1일

클라이언트의 첫 연락을 받고 프로젝트를 시작합니다:

- 요청 내용 파악 (아티스트, 컨셉, 일정, 예산)
- 가능성 검토 (일정 내 완료 가능 여부)
- 견적 및 일정 제안
- 계약 체결

**클라이언트 필요 정보:**
- 아티스트/그룹 정보
- 활동 유형 (무대, 뮤직비디오, 화보 등)
- 원하는 컨셉 또는 레퍼런스
- 필요 날짜
- 예산 범위

### 2단계: 디자인 개발 (Design Development)

**소요 시간:** 5-7일

컨셉을 시각화하고 디자인을 확정합니다:

- 컨셉 리서치 및 분석
- 스케치 제작 (3-5개 옵션)
- 클라이언트 피드백 및 수정
- 최종 디자인 승인
- 소재 선정 및 조달

**클라이언트 역할:** 스케치 검토, 피드백 제공, 최종 승인

### 3단계: 패턴 제작 (Pattern Making)

**소요 시간:** 3-5일

디자인을 기술적 설계도로 변환합니다:

- 치수 수집 또는 확인
- 패턴 제도
- 토일 제작 및 피팅 (필요시)
- 패턴 수정 및 확정
- 기술 사양서 작성

**클라이언트 역할:** 치수 제공, 피팅 참석 (필요시)

### 4단계: 생산 (Production)

**소요 시간:** 7-14일

실제 의상을 제작합니다:

- 재단
- 봉제
- 장식 작업
- 마감 처리
- 품질 검사

**클라이언트 역할:** 대기 (중간 점검 요청 가능)

### 5단계: 배송 (Delivery)

**소요 시간:** 1-2일

완성된 의상을 전달합니다:

- 최종 피팅 및 미세 조정
- 포장
- 배송 또는 수령
- 관리 방법 안내

**클라이언트 역할:** 최종 검수 및 수령 확인

## 일정 관리 팁

### 빠른 진행을 위한 제안

1. **명확한 요구사항**: 처음부터 원하는 것을 명확히 전달하면 수정 횟수가 줄어듭니다
2. **빠른 피드백**: 스케치나 샘플에 대한 피드백이 빠르면 전체 일정이 단축됩니다
3. **치수 정확성**: 정확한 치수를 제공하면 패턴 수정이 최소화됩니다
4. **일정 여유**: 가능하면 필요 날짜에 3-5일 여유를 두세요

### 지연 요인

다음 상황은 일정을 지연시킬 수 있습니다:

- 디자인 결정 지연 (클라이언트 내부 논의 장기화)
- 특수 소재 조달 기간 (해외 수입 소재 등)
- 잦은 디자인 변경
- 피팅 일정 조율 어려움
- 복잡한 장식 작업
