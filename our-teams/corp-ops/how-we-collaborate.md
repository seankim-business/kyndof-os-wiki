# Corp Ops와 협업하는 방법

{: .highlight }
> 재무, HR, IT, 법무 등 Corp Ops 서비스를 효과적으로 이용하는 방법입니다.

## 개요

Corp Ops는 다른 팀들에게 서비스를 제공하는 팀입니다. 급여 처리, IT 지원, 계약 검토, 채용 프로세스 등 - 여러분이 업무에 필요한 기반 서비스를 우리가 담당합니다.

이 문서는 Corp Ops의 각 서비스를 어떻게 요청하고, 무엇을 기대할 수 있는지 안내합니다. 우리와 효과적으로 일하면 여러분의 시간도 절약됩니다.

## 서비스별 요청 가이드

### Finance (재무)

**비용 정산**
- 누구에게: Business Admin Manager
- 제출: 영수증/세금계산서 + 사용 목적
- 정산 주기: 매주 금요일 정산
- 주의: 법인카드 미사용 시 선지출 후 정산. 영수증 분실 시 정산 불가

**예산 확인/요청**
- 누구에게: CTO (예산 관련), Business Admin Manager (잔액 확인)
- 방법: 분기 예산 내 집행은 Slack, 예산 외 지출은 사전 승인 필요
- 필요 정보: 금액, 목적, 예상 ROI (해당 시)

**재무 데이터 요청**
- 누구에게: Business Admin Manager
- 리드타임: 일반 데이터 24시간, 복잡한 분석 3-5일
- 형식: 엑셀 또는 대시보드 접근권

### HR (인사)

**채용 요청**
- 누구에게: Business Admin Manager (프로세스), CTO (승인)
- 제출물: JD(Job Description), 채용 사유, 예상 연봉 범위
- 프로세스: 팀 리더 요청 → CTO/CEO 승인 → 채용 공고 → 서류 스크리닝 → 면접 → 오퍼
- 리드타임: 승인 후 채용까지 통상 4-8주

**온보딩 요청**
- 언제: 입사 확정 후, 입사일 최소 1주 전
- 제출: 이름, 이메일, 소속팀, 필요 시스템/도구 목록
- Corp Ops 준비: 이메일 계정, Slack, Notion 접근권, 필요 SW, 업무 장비

**급여/복리후생 문의**
- 누구에게: Business Admin Manager
- 응답: 24시간 내 (영업일 기준)
- 주의: 개인 급여 정보는 당사자만 문의 가능

### IT

**IT 지원 요청**
- 채널: Slack #it-support 또는 DM
- 긴급: 시스템 다운 등 → 전화/DM
- 포함 정보: 문제 증상, 스크린샷, 시도한 해결책

**새로운 도구/SW 요청**
- 누구에게: CTO
- 제출: 도구 이름, 용도, 비용, 유사 대안 검토
- 프로세스: 보안 검토 → 비용 승인 → 도입

**계정/접근권 요청**
- 누구에게: Business Admin Manager (일반), CTO (관리자 권한)
- 필요 정보: 필요한 시스템, 접근 수준, 사유

### Legal (법무)

**계약 검토 요청**
- 누구에게: CTO (1차), 필요시 외부 법무
- 제출: 계약서 초안, 거래 배경, 우려 사항
- 리드타임: 간단한 계약 2-3일, 복잡한 계약 1-2주

**법적 문의**
- 누구에게: CTO
- 유형: 지적재산권, 고용 관련, 규제 관련 등
- 주의: 중요한 법적 판단은 외부 법률 자문 활용

### GA (총무)

**비품/사무용품 요청**
- 누구에게: Business Admin Manager
- 방법: Slack 또는 공유 요청 양식
- 리드타임: 일반 비품 3-5일, 특수 품목 1-2주

**사무실 관련**
- 누구에게: Business Admin Manager
- 유형: 시설 문제, 청소, 회의실 등
- 긴급: 안전 관련은 즉시 연락

## 커뮤니케이션 채널

| 유형 | 채널 | 예시 |
|------|------|------|
| 일상적 문의 | Slack | 잔액 확인, 간단한 IT 문의 |
| 공식 요청 | Notion 양식 | 채용 요청, 예산 외 지출 |
| 긴급 상황 | 전화/DM | 시스템 장애, 보안 사고 |
| 정기 보고 | 주간 미팅 | 재무 현황, 채용 진행 |

## 정기 프로세스

Corp Ops가 주도하는 정기 프로세스들:

**월간**
- 급여 지급: 매월 25일 (주말 시 전 영업일)
- 재무 마감: 익월 5일까지

**분기**
- 예산 리뷰: 분기 말
- 인사 리뷰: 분기 말

**연간**
- 예산 계획: 12월
- 감사 준비: 회계연도 종료 후
- 인사 제도 리뷰: 1분기

## 협업 시 주의사항

### 미리 요청하세요

Corp Ops의 많은 업무는 리드타임이 필요합니다. 마지막 순간 요청은 품질 저하로 이어집니다.

| 요청 유형 | 권장 리드타임 |
|----------|--------------|
| 채용 시작 | 2주 전 |
| 계약 검토 | 1주 전 |
| 대규모 구매 | 2주 전 |
| 신규 입사자 온보딩 | 1주 전 |

### 완전한 정보를 제공하세요

불완전한 요청은 반복적인 확인으로 시간이 낭비됩니다. 요청 시:
- 목적과 배경을 명확히
- 필요한 마감일을 구체적으로
- 관련 파일/링크를 함께

### 정책을 먼저 확인하세요

자주 묻는 질문의 답은 대부분 정책 문서에 있습니다. 문의 전에:
- 비용 정산 정책
- 휴가 정책
- IT 보안 정책

등을 먼저 확인해주세요.

## FAQ

**Q: 급여가 잘못 입금되었어요**
A: Business Admin Manager에게 즉시 연락하세요. 가능한 빨리 수정합니다.

**Q: 새 노트북이 필요해요**
A: 팀 리더와 먼저 상의 후, CTO에게 요청하세요. 예산과 필요성을 검토합니다.

**Q: 퇴사자 계정은 언제 비활성화되나요?**
A: 퇴사일 당일 모든 시스템 접근이 종료됩니다.

**Q: 개인 장비로 업무해도 되나요?**
A: IT 보안 정책에 따라 제한됩니다. CTO와 상의하세요.
